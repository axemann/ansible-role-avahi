Overview

Ansible Playbook for avahi the zeroconf software.

Role Variables

Available variables are listed below, along with default values (see defaults/main.yml):

hostname: "{{ ansible_hostname }}"
domain: "local"
useipv4: "yes"
useipv6: "no"

Dependencies

None.

Example Playbook

---
- hosts: all
  roles:
    - avahi
